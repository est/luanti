快速开始 ​

5 分钟内从下载到进入游戏。

下载安装 ​

Windows ​

  1. 访问 Luanti 官网 下载 Windows 版
  2. 解压到任意目录，运行 luanti.exe
  3. 在游戏菜单中选择 MineClonia → 点击 "Install Game"

macOS ​

bash
# 使用 Homebrew
brew install luanti

或从 官网 下载 macOS 版。

Linux ​

bash
# Ubuntu / Debian
sudo apt install luanti

# Arch Linux
sudo pacman -S luanti

# Flatpak (通用)
flatpak install flathub org.luanti.luanti

Android ​

在 Google Play 或 F-Droid 搜索 "Luanti" 下载。

安装 MineClonia ​

启动 Luanti 后：

  1. 点击 Content (内容) 标签
  2. 点击 Browse Online Content (浏览在线内容)
  3. 搜索 MineClonia
  4. 点击 Install (安装)
  5. 返回主菜单，点击 Start Game (开始游戏)
  6. 在游戏列表中选择 MineClonia，点击 Play (游玩)

国内用户

如果在线内容下载慢，可以手动安装：

  1. 从 GitHub Releases 下载最新版
  2. 解压到 Luanti 的 games/ 目录下
  3. 重启 Luanti 即可看到 MineClonia

第一个夜晚 ​

白天要做的事 ​

  1. 撸树 — 和 MC 一样，长按左键挖原木
  2. 做工作台 — 4 个木板合成 1 个工作台
  3. 做木镐 — 在工作台上用木棍 + 木板合成
  4. 挖石头 — 用木镐挖圆石
  5. 做石制工具 — 石镐、石剑、石斧

天黑前 ​

  • 挖一个山洞或建一个小房子
  • 做一扇木门
  • 用煤炭 + 木棍做火把
  • 做一张床（3 个羊毛 + 3 个木板）— 可以跳过夜晚

第一晚生存 ​

如果没来得及做床：

  • 待在室内，不要开门
  • 用火把照亮周围
  • 挖矿向下发展

注意

MineClonia 的怪物行为和 MC 略有不同，详见 生物图鉴。

基本操作 ​

操作键位

切换飞行K（创造模式）
