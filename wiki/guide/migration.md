# 从 Minecraft 迁移

已经在玩 Minecraft？这张表帮你 5 分钟搞懂 MineClonia 的差异。

## 速查对照表

### 基本差异

| 项目 | Minecraft | MineClonia |
|------|-----------|------------|
| 价格 | ¥165 (Java) / 免费 (基岩) | **完全免费** |
| 引擎 | 自研 | Luanti (开源) |
| Mod 支持 | Forge/Fabric | 原生 Lua Modding |
| 最大地图 | 6000万×6000万 | **无限**（理论值） |
| 红石 | Redstone | **不存在**，用 mesecons 替代 |
| 附魔 | 附魔台 | 附魔台（机制略有不同） |
| 酿造 | 酿造台 | 酿造台（配方基本一致） |
| 下界 | Nether | Nether |
| 末地 | End | **不存在** |

### 物品名称对照

| Minecraft | MineClonia | 备注 |
|-----------|------------|------|
| 钻石 | Diamond | 一致 |
| 铁锭 | Iron Ingot | 一致 |
| 红石粉 | Mese Crystal | **名称不同，功能类似** |
| 红石火把 | Mesecon Torch | 对应物 |
| 活塞 | Piston | 对应物 |
| 命令方块 | Command Block | 需要权限 |
| 附魔之瓶 | XP Bottle | 一致 |

### 机制差异

#### 红石 → Mesecons

MineClonia 用 **Mesecons** 替代红石系统：

- Mese Crystal（美赛水晶）= 红石粉
- Mesecon Torch = 红石火把
- 逻辑门、活塞、中继器等都有对应物
- 布线规则和红石**不完全相同**，需要重新学习

::: tip 建议
如果你是红石玩家，先在创造模式里玩一下 Mesecons，
理解信号传播规则后再搞复杂电路。
:::

#### 附魔

- 附魔台存在，但附魔等级和效果与 MC 有差异
- 需要青金石（Lapis Lazuli）作为附魔材料
- 附魔书可以通过钓鱼获得

#### 酿造

- 酿造台、药水瓶、材料基本一致
- 部分药水效果有差异
- 烈焰粉仍然是燃料

#### 没有的东西

以下是 MC 有但 MineClonia **没有**的内容：

- ❌ 末地 / 末影龙
- ❌ 潜影盒
- ❌ 鞘翅 / 烟花火箭飞行
- ❌ 劫掠 / 村庄英雄
- ❌ 潮涌核心
- ❌ 潜声传感器
- ❌ 细雪

#### 多出来的东西

MineClonia 有一些 MC **没有**的内容：

- ✅ 僵尸围城（Zombie Siege）系统
- ✅ 袭击（Raid）系统
- ✅ 更丰富的天气系统
- ✅ 内置的成就系统
- ✅ 内置的物品百科（按 I 键查看）

## 创造模式

和 MC 一样有创造模式：

- 按 `K` 切换飞行
- 物品栏包含所有方块和物品
- 可以直接放置任何方块

## 服务器

MineClonia 的多人游戏基于 Luanti 服务器：

- 主菜单 → **Join Game** (加入游戏)
- 可以直接输入服务器 IP 连接
- 也可以浏览公共服务器列表

::: info 国内服务器
国内 MineClonia 服务器较少，可以：
- 自己开服（Luanti 自带服务器功能）
- 在 Luanti 社区寻找中文服务器
:::

## 常见问题

### Q: 存档能通用吗？

**不能。** Minecraft 和 MineClonia 的存档格式完全不同。

### Q: Mod 能通用吗？

**不能。** 两个游戏的 Mod 系统完全不同。但 MineClonia 有自己的 Mod 生态。

### Q: 材质包能用吗？

**不能直接用。** 但 MineClonia 支持自定义材质包，格式与 MC 不同。

### Q: 操作习惯能保持吗？

**基本可以。** WASD 移动、空格跳跃、左键挖右键放都一样。背包和合成界面略有差异。
