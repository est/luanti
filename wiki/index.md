## 什么是 MineClonia？

MineClonia 是一款基于 [Luanti](https://www.luanti.org/) 引擎的开源生存沙盒游戏，
玩法与 Minecraft 高度相似，但**完全免费、开源**。

- 🆓 **免费** — 不需要购买，下载即玩
- 🌍 **开源** — 代码完全公开，社区驱动开发
- 🔧 **可 Mod** — 丰富的 Mod 生态，自由扩展
- 💻 **跨平台** — Windows / macOS / Linux / Android 全平台支持
- 🪶 **轻量** — 低配电脑也能流畅运行

## 数据来源

本 Wiki 的物品、合成表、生物等数据**直接从 MineClonia 源码提取**，
每次游戏更新后自动同步，确保信息准确无误。

::: tip 数据准确性
所有技术数据（硬度、爆炸抗性、伤害值等）均来自源码中的注册定义，
不是人工录入，不会有过时或错误的信息。
:::
